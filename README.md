## Tentang Aplikasi
Aplikasi Kalkulator BMI merupakan aplikasi untuk menghitung bobot ideal seseorang. Perhitungan bobot ideal menggunakan rumus

````bash
BMI = berat badan / (tinggi badan)^2
````
Keterangan:
`Berat Badan (kg) ; Tinggi Badan (m)`

Dari perhitungan tersebut akan didapatkan hasil dan dikategorikan seperti berikut:

````bash
BMI < 18.5          ==> Berat Badan Kurang
18.5 ‚â§ BMI ‚â§ 22.9   ==> Berat Badan Normal
23 ‚â§ BMI ‚â§ 24.9     ==> Berat Badan Berlebih
25 ‚â§ BMI ‚â§ 29.9     ==> Obesitas 1
BMI > 30            ==> Obesitas 2
````

Aplikasi Kalkulator BMI dibuat dengan menggunakan `bash scripting`, di mana aplikasi ini memiliki 3 menu utama yaitu:

````bash
1. Cek BMI Tiap Siswa
   -> Menghitung, menampilkan, dan menyimpan BMI dari tiap siswa
2. Cek BMI Seluruh Siswa
   -> Menghitung, menampilkan, dan menyimpan BMI dari beberapa siswa sekaligus
3. Tampilkan Data Keseluruhan
   -> Menampilkan statistik BMI siswa secara keseluruhan
````
